# 🧪 Lab 3 -- React Student Scoreboard

A React application that displays and updates student scores using a
component-based architecture.\
This project demonstrates the use of state, props, reusable components,
conditional rendering, and Virtual DOM updates.

------------------------------------------------------------------------

## 📚 Objective

To build a dynamic student scoreboard that:

-   Displays student details and scores
-   Updates UI in real time using React state
-   Uses reusable components
-   Applies conditional styling based on pass/fail status

------------------------------------------------------------------------

## ⚛️ Built With

-   React (with Vite)
-   JSX
-   Functional Components
-   Tailwind CSS / CSS Modules

------------------------------------------------------------------------

## ✨ Key Features

-   Component-based architecture (Header, Table, Score Card, Student
    Row)
-   Dynamic rendering of student data
-   Real-time UI updates using state
-   Data passed from parent to child using props
-   Conditional rendering for Pass / Fail status
-   Conditional styling based on student score
-   Fast updates using React Virtual DOM

------------------------------------------------------------------------

## 🧠 Practice Concepts

-   Creation of reusable components
-   Props for data transfer
-   State management for dynamic UI
-   Conditional rendering
-   Code refactoring for better readability

## ⚙️ Installation & Setup

1.  Clone the repository

    git clone
    https://github.com/your-username/react-student-scoreboard.git

2.  Navigate to the project

    cd react-student-scoreboard

3.  Install dependencies

    npm install

4.  Run the development server

    npm run dev

------------------------------------------------------------------------

## 🖥️ How It Works

-   Student data is stored in React state
-   Passed to child components using props
-   Each student is rendered using a reusable row component
-   Pass/Fail status is shown using conditional rendering
-   Styles change dynamically based on score

------------------------------------------------------------------------

## 🎯 Conditional Logic

-   Score ≥ Passing Marks → Pass
-   Score \< Passing Marks → Fail
